# AI-Based System for Grammar Correction and Emotion Detection in Speech

## 📌 Overview

This project is an AI-powered system that analyzes speech input to:

* Convert speech to text
* Detect emotions from speech signals
* Perform grammar and punctuation correction
* Provide real-time feedback and visualization

---

## 🚀 Features

* 🎤 Speech-to-Text Conversion
* 😊 Emotion Recognition (Deep Learning Model)
* ✍️ Grammar & Punctuation Correction
* 📊 Real-time Feedback Interface

---

## 🛠 Tech Stack

* Python (Flask)
* Node.js (Frontend Server)
* Deep Learning (PyTorch)
* NLP Libraries

---

## 📂 Project Structure

```
app.py              # Backend (Flask)
server.js           # Frontend server
templates/          # UI files
requirements.txt    # Python dependencies
package.json        # Node dependencies
setupffmpeg.py 
```

---

## ⚙️ Setup Instructions

### 1. Clone the repo

```
git clone https://github.com/SHASHI-29/AI-BASED-EMOTION-AND-GRAMMAR-SYSTEM.git
cd AI-BASED-EMOTION-AND-GRAMMAR-SYSTEM
```

### 2. Install dependencies

#### Python:

```
pip install -r requirements.txt
```

#### Node:

```
npm install
```

---

## ▶️ Run the Project

```
python app.py
node server.js
```
